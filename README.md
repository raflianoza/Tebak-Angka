# Tebak Angka
<p>Tebak Angka adalah sebuah game dimana kita menebak 3 digit dari angka ratusan</p>

<h2>Download Game</h2>
<code>git clone https://github.com/raflianoza/Tebak-Angka.git</code>

<br>

<h2>Run the Game</h2>

<h3>Macos / Linux</h3>
<code>python3 main.py</code>

<h3>Windows</h3>
<code>python main.py</code>
